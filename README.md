## Getting Started

To get a local copy up and running follow these steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/fabianskutta/error-pages.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. run dev environment
   ```sh
   npm run dev
   ```
